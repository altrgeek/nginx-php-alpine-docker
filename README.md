# php-nginx-alpine

Lightweight php nginx server on an alpine image


## Features
<!------------------------------------------------------->

* Under 35 MB
* Alpine 3.17
* PHP 8.1.13-r0
* Nginx 1.22.1-r0


## Installation
<!------------------------------------------------------->

```sh
docker run --name some-php-nginx-alpine -d -v /your/php/app:/app/www -p 8888:8888 altrgeek/nginx-php-alpine:latest
```


## Dependencies
